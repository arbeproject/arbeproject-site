# ARBE Project – Official Site

This repository contains the official website for the ARBE Project (Algorithmic Resource‑Based Economy) and the AMN sovereign digital currency.

## Tech stack

- [Astro](https://astro.build/) for the static site
- Deployed on [Netlify](https://www.netlify.com/)
- Source hosted on GitHub

## Getting started

1. Install dependencies:

   ```bash
   npm install
   ```

2. Run the dev server:

   ```bash
   npm run dev
   ```

3. Build for production:

   ```bash
   npm run build
   ```

## Structure

- `/src/pages/index.astro` — Landing page
- `/src/pages/docs` — General documentation entry
- `/src/pages/whitepaper` — Whitepaper and technical papers
- `/src/pages/protocols` — Protocol specs (SLP, RTM, etc.)
- `/src/pages/slp` — Sovereign Ledger Protocol details
- `/src/pages/rtm` — Resource Token Model
- `/src/pages/ama` — Algorithmic Monetary Authority (AMA)
- `/src/pages/legal` — Legal texts (Decree‑Law, annexes, licenses)

You can edit each section as Markdown/HTML/MDX inside the corresponding folder.
